# Επίσημες πηγές διοικητικών ορίων Ελλάδας (ενδεικτικές)

## Προτεινόμενες κρατικές/θεσμικές πηγές
- Εθνική Υποδομή Γεωχωρικών Πληροφοριών (geodata.gov.gr)
- Ελληνική Στατιστική Αρχή (ΕΛΣΤΑΤ) — θεματικά/διοικητικά όρια μέσω geodata
- Κυβερνητικός Χάρτης / geoportal (maps.gov.gr)
- ΥΠΕΝ GIS portals (π.χ. gis.ypeka.gr)

## Σημείωση
- Η πρόσβαση/λήψη απαιτεί έλεγχο της εκάστοτε άδειας και του διαθέσιμου format (WMS/WMTS/GeoJSON/Shapefile).
